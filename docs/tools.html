CLI Security Tools

14 Python scripts for penetration testing and security auditing

Website Security Checker

Scans HTTP security headers (HSTS, CSP, X-Frame-Options, XSS-Protection), detects web technologies, and checks for common misconfigurations.

Usage: python3 web-check.py example.com

Output: JSON with headers, tech stack, vulnerabilities

Port Scanner

Multi-threaded TCP port scanner. Scans 25 common ports including HTTP, HTTPS, SSH, FTP, SMTP, DNS, MySQL, PostgreSQL, and Redis.

Usage: python3 port-scan.py 10.0.0.1

Output: JSON with open ports and service detection

SSL Certificate Inspector

Retrieves and analyzes SSL/TLS certificates. Shows issuer, subject, validity period, SANs, and cipher suite information.

Usage: python3 ssl-check.py example.com

Output: JSON with full certificate details

DNS Lookup Tool

Performs DNS queries for A, AAAA, MX, NS, TXT, CNAME, SOA records. Supports custom DNS servers.

Usage: python3 dns-lookup.py example.com

Output: JSON with all record types

HTTP Header Inspector

Examines HTTP security headers. Checks for HSTS, CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, and Permissions-Policy.

Usage: python3 http-headers.py example.com

Output: JSON with header analysis

Subdomain Finder

Discovers subdomains using 50+ common patterns. Tests each subdomain for DNS resolution and HTTP response.

Usage: python3 sub-find.py example.com

Output: JSON with discovered subdomains

Log Analyzer

Analyzes web server access logs. Detects SQL injection attempts, XSS attacks, brute force login attempts, and path traversal.

Usage: python3 log-analyzer.py access.log

Output: JSON with attack patterns found

SSL Cert Watcher

Monitors SSL certificate expiry for multiple domains. Shows days remaining, issuer, and warns about upcoming expiration.

Usage: python3 cert-watch.py example.com google.com github.com

Output: JSON with expiry status per domain

Password Strength Auditor

Evaluates password strength based on length, character variety, entropy, and common pattern detection.

Usage: python3 pass-audit.py MyP@ssw0rd123

Output: JSON with strength score and analysis

File Integrity Monitor

Monitors files and directories for changes using SHA-256 hashing. Detects new, modified, and deleted files.

Usage: python3 file-monitor.py /path/to/watch

Output: JSON with file status changes

WHOIS Domain Lookup

Retrieves domain registration information via RDAP. Shows registrar, creation/expiry dates, name servers, and registrant contact.

Usage: python3 whois-lookup.py example.com

Output: JSON with WHOIS/RDAP data

HTTP Response Benchmarker

Benchmarks HTTP response times. Runs configurable requests and reports min, max, average, and standard deviation.

Usage: python3 http-bench.py https://example.com

Output: JSON with timing statistics

URL Extractor

Extracts all URLs from a web page. Parses HTML and finds links, scripts, images, stylesheets, and other resources.

Usage: python3 url-extract.py https://example.com

Output: JSON with all discovered URLs

File Hash Checker

Computes file hashes using MD5, SHA-1, SHA-256, and SHA-512. Supports single files and batch processing.

Usage: python3 hash-check.py file.txt

Output: JSON with all hash values
