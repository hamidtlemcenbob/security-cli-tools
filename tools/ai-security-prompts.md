# AI Security Prompts Pack
Professional prompts for ChatGPT, Claude, and AI assistants

## Category 1: Vulnerability Analysis
---
Prompt 1: Analyze this web application source code for common vulnerabilities (SQLi, XSS, CSRF, SSTI, IDOR). Provide CWE references and CVSS scores.

Prompt 2: Review this API endpoint for authentication bypass, rate limiting issues, and mass assignment vulnerabilities.

Prompt 3: Analyze this smart contract for reentrancy, overflow, and access control vulnerabilities.

Prompt 4: Given these HTTP response headers, identify missing security headers and their potential impact.

## Category 2: Reconnaissance
---
Prompt 5: Given a domain name, list all subdomain enumeration techniques including passive (Certificate Transparency, DNS dumpster) and active (brute force with common wordlist).

Prompt 6: Generate a comprehensive nmap command for full port scan, service version detection, and OS fingerprinting with evasion techniques.

Prompt 7: Extract all endpoints, parameters, and API routes from this JavaScript bundle file.

## Category 3: Exploitation
---
Prompt 8: Given a SQL injection point in a parameter, generate UNION-based, error-based, and blind SQLi payloads for MySQL, PostgreSQL, and MSSQL.

Prompt 9: Analyze this XSS context and generate appropriate payloads for stored, reflected, and DOM-based XSS with WAF bypass techniques.

Prompt 10: Given a JWT token, analyze its structure, check for common vulnerabilities (none algorithm, weak secret, KID injection).

## Category 4: Post-Exploitation
---
Prompt 11: Given Linux privilege escalation checklist: enumerate SUID binaries, cron jobs, kernel exploits, sudo permissions, and capabilities.

Prompt 12: Generate a reverse shell one-liner for Linux/Windows that bypasses common AV solutions.

## Category 5: Reporting
---
Prompt 13: Generate a professional bug bounty report template with executive summary, vulnerability details, proof of concept, CVSS scoring, remediation steps.

Prompt 14: Convert these scan results into a client-ready penetration testing report with risk ratings and business impact analysis.

## Category 6: Automation
---
Prompt 15: Generate a Python script that automates subdomain takeover detection using DNS resolution and HTTP response fingerprinting.

Prompt 16: Create a bash script for automated reconnaissance pipeline: subdomain discovery -> port scanning -> screenshot -> vulnerability scanning.

## Category 7: Reverse Engineering
---
Prompt 17: Analyze this decompiled Java class for hardcoded credentials, API keys, and security-sensitive logic.

Prompt 18: Given an obfuscated JavaScript, deobfuscate and extract the core logic.

## Category 8: OSINT
---
Prompt 19: Given an email address, outline the OSINT gathering process including social media, data breaches, and professional network searches.

Prompt 20: Given a company name, generate a complete OSINT reconnaissance plan covering technology stack, employees, infrastructure, and third-party services.
